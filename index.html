Задача №9: Анализ координат

Условие: В файле даны координаты двух точек $(x_1, y_1)$ и $(x_2, y_2)$. Найдите количество пар, образующих отрезок, который не пересекает ни ось X, ни ось Y.

💡 Логика решения

Отрезок не пересекает оси координат только тогда, когда обе точки находятся в одной четверти. Это означает, что знаки соответствующих координат должны совпадать.

Условие: (Знак x₁ == Знак x₂) И (Знак y₁ == Знак y₂)

2. Решение в Excel: Функция ЗНАК

Для проверки совпадения знаков удобно использовать функцию ЗНАК(number) (в английской версии SIGN).

Как работает функция:

  • Если число положительное (>0) → возвращает 1
  • Если число отрицательное (<0) → возвращает -1
  • Если число равно 0 → возвращает 0

Если знаки двух чисел одинаковы, то и результат функции ЗНАК для них будет одинаковым.

Число A Число B ЗНАК(A) ЗНАК(B) Совпадают? Пересекает ось?
5 3 1 1 ✅ Да Нет
-2 -5 -1 -1 ✅ Да Нет
4 -2 1 -1 ❌ Нет Да (ось Y)
0 5 0 1 ❌ Нет Да (касается)

3. Решение на Python (Без библиотек)

Этот скрипт читает файл построчно (как текст), разбивает строку на части и проверяет условие. Он не требует установки pandas и работает быстро.

Примечание: Скрипт ожидает, что числа в строке разделены пробелами или табуляцией.

# Открываем файл для чтения
filename = '9-132.txt'  # Или .csv, если сохраните как текст
count = 0

try:
    with open(filename, 'r', encoding='utf-8') as f:
        for line in f:
            # Убираем лишние пробелы по краям и разбиваем строку
            # split() без аргументов делит по любым пробелам/табуляциям
            parts = line.strip().split()
            
            # Проверка: должно быть ровно 4 числа
            if len(parts) != 4:
                continue
            
            # Преобразуем строки в целые числа
            x1 = int(parts[0])
            y1 = int(parts[1])
            x2 = int(parts[2])
            y2 = int(parts[3])
            
            # Логика проверки:
            # 1. Знаки X совпадают (произведение > 0)
            # 2. Знаки Y совпадают (произведение > 0)
            if (x1 * x2 > 0) and (y1 * y2 > 0):
                count += 1

    print(f"Ответ: {count}")

except FileNotFoundError:
    print("Файл не найден! Проверьте имя файла.")
except ValueError:
    print("Ошибка в данных: проверьте, что в файле только целые числа.")
